# Alternet Explorer
Полностью графический браузер для сети АльтерНет

## Быстрый старт (Debian/Astra/Mint/Ubuntu)  
```
sudo apt install python3-commonmark python3-pygame python3-pyperclip -y
git clone https://github.com/Michaelionin/ae.git ae
cd ae
python3 ae.py
```
Либо скачайте бинарники!
